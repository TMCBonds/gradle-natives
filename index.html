Gradle Natives Plugin

Gradle plugin to aid in managing native libraries associated with Java-based projects.

Welcome

This Gradle plugin fills the native library support gap and provides a simple means of unpacking native libraries provided by your project dependencies into a well-known and usable location. Configuration is simple; just add the following to your `gradle.build` file:


buildscript {
    repositories {
        maven {
            url 'http://dl.bintray.com/cjstehno/public/'
        }
    }

    dependencies {
        classpath 'gradle-natives:gradle-natives:0.1'
    }
}

apply plugin:'natives'
And then configure the dependency libraries you want searched by adding a configuration block, such as: 

natives {
    jars = [ 'lwjgl-platform-2.9.1-natives-windows' ]
    platforms = 'windows'
}
Also, I have written a blog post, "[Going Native with Gradle](https://github.com/cjstehno/coffeaelectronica/wiki/Going-Native-with-Gradle)", which goes into a bit more detail on how to get up and running.
